## 搬瓦工是什么？先把基本概念说清楚

搬瓦工，英文名 BandwagonHost，缩写 BWG，是加拿大 IT7 Networks 旗下的 VPS 主机商。之所以叫"搬瓦工"，是因为 BandwagonHost 的发音念出来有点像 bān wǎ gōng，就这么叫开了。

它做的是 KVM 架构的虚拟服务器（VPS）。简单说，你买了之后拿到一台独立的 Linux 服务器，有独立 IP，有 root 权限，想装什么装什么——建站、学 Linux、跑程序、挂 bot、部署 AI 应用，都行。官方不管你的软件层，你自己负责。这也是为什么价格比共享主机便宜得多的原因。

搬瓦工最早做到口碑的，是它的 CN2 GIA 网络线路。这条线路是中国电信的高端国际专线，国内访问速度快、丢包率低，用过的人都说这一点比普通 VPS 差距明显。
